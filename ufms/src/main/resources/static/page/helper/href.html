产品反馈首页地址
URL（点击复制）：
参数 类型 说明
pid Long 产品ID（必选）
uid String 用户ID，由开发者自行维护，必须为UUID，不传入则表示随机生成
uname String 用户名，由开发者自行维护，不传入则表示随机生成
uavatar String 用户头像url，由开发者自行维护，不传入则一直为空，即使以后传入也不会生效

注意：
uid、uname两者必须同时传入，否则自定义登录态会构建失败

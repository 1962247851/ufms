项目反馈首页地址
URL（点击复制）：https://localhost:1116/page/product/feedback.html?pid=
参数 说明
pid 产品ID（必选）
uid 用户ID，由开发者自行维护，不传入则表示随机生成
uname 用户名，由开发者自行维护，不传入则表示随机生成
uavatar 用户头像url，由开发者自行维护，不传入则为空

注意：
uid、uname、uavatar三者必须同时传入，否则自定义登录态会构建失败
